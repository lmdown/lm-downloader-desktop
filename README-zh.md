# 关于源码停止开放的公告

近期收到多位用户反馈，有不法分子利用本项目开源代码进行二次包装，并在未授权的情况下通过第三方平台非法售卖，且对用户进行欺诈。为了防止此类事件再次发生，保护用户权益，开发者决定：

- 停止源码公开： 本项目即日起由开源转为闭源维护。
- 唯一官方渠道： 本仓库的 Releases 页面和官方网站为唯一可信下载源。
- 法律追责： 我们已保留相关侵权证据，并保留追究法律责任的权利。
